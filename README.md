Приложение Django для создания и отображения древовидного меню, редактируемого через стандартную админку. Меню хранится в базе данных, активный пункт определяется по URL текущей страницы, и на отрисовку меню требуется всего 1 запрос к БД.  

## Что реализовано

- Древовидное меню с неограниченной вложенностью.
- Раскрытие всех пунктов выше активного и первого уровня вложенности активного.
- Поддержка нескольких меню на одной странице (по названию).
- URL пунктов меню можно задавать явно или через named url.
- Редактирование меню через стандартную админку Django.
- Отображение меню через template tag `{% draw_menu 'menu_name' %}`.
- Меню можно использовать на любой странице сайта.
- Контейнеризация проекта через Docker для удобного запуска.

## Установка и запуск (Docker)

1. Клонируем репозиторий:

```bash
git clone <URL_репозитория>
cd <название_репозитория>

2. Сборка Docker-контейнера:

docker build -t django-menu-app .


3. Запуск приложения:

docker run -p 8000:8000 django-menu-app

## Установка и запуск через GitHub (без Docker)

1. Клонируем репозиторий:

git clone <URL_репозитория>
cd <название_репозитория>

2. Создаём и активируем виртуальное окружение:

python -m venv venv
source venv/bin/activate   # Linux/macOS
venv\Scripts\activate      # Windows

3. Устанавливаем зависимости:
pip install -r requirements.txt


4. Выполняем миграции:
python manage.py migrate
Создаём суперпользователя для админки:

5. Создаём суперпользователя для админки:
python manage.py createsuperuser

6. Запускаем сервер:
python manage.py runserver
